

LanBox
---

**写在前面**  

此程序主要为 minipc(cubieboard、树莓派等) 而设计，而且要有一台无线路由器。  
我们经常在手机或者平板电脑上存放视频和图片，它们占用了大量的空间。  
设想如果数据放在一台接入家庭局域网的 minipc 上，通过 LanBox 进行共享。  
手机或平板可以通过浏览器访问 minipc 来播放视频和查看图片。  
现在这个项目刚刚开始。  


**特性**  

* 尽量使用html5来播放视频，不支持时会使用 Flash  


**计划**

* 支持看图  
* 能够访问压缩包中的数据  
* 文件标签和评分  
* 文件管理
* 界面美化
